# vue-music
项目概述: 该项目是基于vue2仿造网易云音乐PC端，是我自己个人独自开发。实现内容：歌单模块，我的喜欢，歌曲播放控制台，歌词，搜索模块,个人中心，歌手模块，收藏模块，排行榜...等

技术选型: Vue2+Vuex+Axios+elementUI+Cropper
